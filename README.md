# Система импорта Базы Данных о продажах, заказах, складах, доходах

## Описание функционала
Функционал реализован контроллером [ImportController.php](app/Http/Controllers/api/v1/ImportController.php), который находится в папке `app/Http/Controllers/api/v1/`.

В контроллере метод `importData()` выполняет автоматизированный процесс импорта данных из внешнего API в базу данных пользователя MySQL с обработкой пагинации.

Для Базы MYSQL созданы миграции

Для сущностей Склады загрузка данных только за текущий день

## Маршрутизация
Роуты находятся в файле `app/routes/api.php`.

Доступны следующие Роуты:

* <span style="color: rgb(25, 220, 55);"> /api/v1/import/sales </span> - загружает данные по Продажам.
* <span style="color: rgb(25, 220, 55);"> /api/v1/import/orders </span> - загружает данные по Заказам
* <span style="color: rgb(25, 220, 55);"> /api/v1/import/stocks </span> - загружает данные по Складам
* <span style="color: rgb(25, 220, 55);"> /api/v1/import/incomes </span> - загружает данные по Доходам

## Назначение
Метод предназначен для:
* загрузки данных о продажах, заказах, складах, доходах за указанный период из внешнего API;
* обработки пагинации (множества страниц данных);
* сохранения всех записей в локальной базе данных;
* отслеживания прогресса и обработки ошибок.
